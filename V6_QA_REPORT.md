## สิ่งที่ปรับใน V6
1. ปรับบทเรียนรายชั่วโมงทั้งหมดให้ใช้เว็บสอนจบในคาบได้จริง
2. เพิ่มความหมายเฉพาะบท เหตุผลในการเรียน และคำสำคัญ
3. เพิ่มตัวอย่างทดลองสอน 4 ระดับก่อนใบงาน
4. เพิ่ม Teacher Script และคำถามนำที่ครูใช้พูดได้ทันที
5. ปรับใบงานให้ตอบพร้อมเหตุผล ไม่ใช่แค่เติมคำตอบ
6. ปรับเกมเป็น Interactive Quiz พร้อมเฉลยเหตุผลทุกชั่วโมง
7. ปรับ CSS สำหรับหน้าจอใหญ่/iPad/มือถือ ลดการทับกันของปุ่ม เมนู และ footer
