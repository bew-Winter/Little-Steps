# Little Steps V2 Ready-to-Teach V6

เวอร์ชันนี้ปรับบทเรียนให้เป็นแบบใช้เว็บสอนจบในคาบได้จริง เด็กใช้เพียงสมุดจดบันทึกและดินสอ เนื้อหาทุกชั่วโมงมีความหมาย เหตุผล ตัวอย่างหลายระดับ ใบงานที่ต้องอธิบายเหตุผล เกมทบทวน และเฉลยครู

## จุดสำคัญของ V6
- ปรับไฟล์รายชั่วโมงทั้งหมดให้ไม่เป็นแค่หัวข้อหรือคำสั่ง
- ทุกบทมี Teacher Script และคำถามนำ
- ทุกบทมีตัวอย่าง 4 ระดับก่อนใบงาน
- ทุกใบงานต้องตอบพร้อมเหตุผล
- เกมทุกชั่วโมงเป็น Interactive Quiz พร้อมเฉลยเหตุผล
- ไม่บังคับใช้สื่อภายนอก หากครูมีรูป/ของจริงใช้เสริมได้เท่านั้น
- ปรับ CSS ลดปัญหาปุ่ม เมนู footer หรือ sidebar ทับเนื้อหา

เปิดใช้งานจาก `index.html`
