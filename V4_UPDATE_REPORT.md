# Little Steps V2 Refined Content + Games V4

## สิ่งที่ปรับรอบนี้
- ปรับเนื้อหาไฟล์รายชั่วโมงทั้งหมด 104 ไฟล์ ให้มีช่วงอธิบายความหมาย เหตุผล ตัวอย่างหลายระดับ ขั้นสอนจริง ใบความรู้ ใบงาน เกม Exit Ticket และแนวตรวจครูชัดขึ้น
- เพิ่มระบบเกม V4 แบบ interactive quiz ในทุกชั่วโมง มีคะแนน เฉลยเหตุผล ปุ่มเล่นใหม่ และข้อถัดไป
- ปรับใบงานให้มี 3 ตอน: เช็กความเข้าใจ / ลงมือทำ / อธิบายเหตุผล เพื่อไม่ให้เด็กทำตามคำสั่งโดยไม่เข้าใจ
- เพิ่ม Teacher Script และคำถามนำ เพื่อให้ครูใช้ทดลองสอนก่อนปล่อยทำงานเดี่ยว
- ปรับ CSS ให้หน้าสะอาดขึ้น ลดการทับซ้อนของปุ่ม เกม ใบงาน และเนื้อหายาว
- ยังคงระบบโหมดครู/นักเรียน เมนูมือถือ Progress และ Print/PDF จาก V3

## แนวคิดสำคัญ
ทุกชั่วโมงต้องตอบได้ 3 คำถามก่อนทำใบงาน:
1. เรื่องนี้คืออะไร
2. ทำไมต้องเรียนเรื่องนี้
3. รู้ได้อย่างไรว่าคำตอบ/วิธีคิดถูกต้อง

## วิธีเช็กหลังเปิดเว็บ
1. เปิด index.html
2. เข้าแต่ละวิชา → เลือกชั่วโมง
3. ลองหน้าเกม กดคำตอบ แล้วกดข้อถัดไป
4. เปิดมือถือหรือย่อหน้าจอ เพื่อดูเมนูบทเรียนไม่ทับเนื้อหา
5. กดพิมพ์/PDF เพื่อดูหน้ากระดาษ
